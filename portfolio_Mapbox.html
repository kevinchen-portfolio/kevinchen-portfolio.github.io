About

This Mapbox web map displays information about particular murals in Chicago.

The dataset used to create this map was processed in ArcGIS Pro and converted into a GeoJSON file using QGIS. After being processed, the dataset was imported and edited in Mapbox Studio. The basemap was customized on Mapbox Cartogram. Users can click the icons to view the mural's name, artist(s), location, and image.

Application
