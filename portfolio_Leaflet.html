About

This web map depicts Stop of Interest signs, parks and trails in British Columbia. It was built using Leaflet. The data was downloaded from British Columbiaâ€™s Open Data Portal, which was converted to a GeoJSON file using QGIS. Then it was built using JavaScript objects and linked to the Leaflet CDN.

Five Leaflet plugins have been embedded and customized in the map:

  • Marker Cluster: employs marker clustering functionality to increase user's interactivity.
  • Full Screen: a Html5 full-screen plugin.
  • Better Scale: provides map readers with an alternating scale bar.
  • Mouse Position: displays the mouse pointer's geographic coordinates as it moves over the map.
  • Double Right-Click Zoom: a function that allows map readers to zoom-out of the map by double right-clicking the mouse.
