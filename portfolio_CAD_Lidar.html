About

This project involved the importation of aerial LiDAR data into AutoCAD. The work process included the following key steps:

  • Importing Aerial LiDAR data into AutoCAD drawings from LAS files.
  • Creating Surface from point cloud.
  • Expanding the ground surface and definition nodes.
  • Creating alignment from objects and profile from Surface.

Application

  • AutoCAD Civil 3D 2019
  • AUTODESK ReCap

Data

  • NSCC's Applied Geomatics Research Group
