About

This project involved the registration and digitization of an image in MapInfo Pro. The work process began with the selection of control point coordinates on an aerial photograph. Next, the the registration function was used to line up the road vector data and the raster image.

The third part of the process involved digitizing features of the aerial photograph. The cosmetic layer was used to capture features from the raster image and convert them to geographic data, which included building footprints, roads, trees, and rivers, as shown on the map.

Application

  • MapInfo Pro Version 2019.3
